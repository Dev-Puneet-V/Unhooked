# Unhooked Mobile App
Unhooked is a ``AI-first`` current affairs quiz platform. Get everyday quiz based on the quizzes on the basis of current day affairs. Play the quiz, score well consitently and win prize without a single penny of investment. 

## Version
v1.0.0 - Initial Release

## How to run
- `npm i`
- `npm run android`
