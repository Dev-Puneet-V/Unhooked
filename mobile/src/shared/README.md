# Shared

Reusable pieces used by multiple features:

| Folder | Purpose |
|--------|---------|
| `components/` | Buttons, inputs, cards, layout |
| `hooks/` | Cross-feature React hooks |
| `utils/` | Pure helpers |
| `constants/` | App-wide constants |
| `api/` | HTTP client and shared API helpers |
| `theme/` | Colors, spacing, typography |
| `types/` | Shared TypeScript types |

Keep feature-specific code under `src/features/<name>/`.
