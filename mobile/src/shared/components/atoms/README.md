# Features

Put each product area in its own folder:

| Folder | Purpose |
|--------|---------|
| `auth/` | Login, register, session |
| `onboarding/` | First-run / welcome flow |
| `quiz/` | Daily quiz gameplay |
| `profile/` | User profile and stats |
| `prizes/` | Prizes / leaderboard |

Inside a feature, add `screens/`, `components/`, `hooks/`, or `api/` as needed. Anything reused across features belongs under `src/shared/`.
